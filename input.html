Input





W/A/S/D - Movement

Spacebar - Jump or Fly up

Left Shift - Fly down

Mouse - View

Left button - Destroy block or Take objects in inventory

Right button - Place block or Put objects in inventory

Right button + Left Shift - Put 1 object in inventory

Enter - Save the world(OLD)

C - Save position(OLD)

R - Return to saved position(OLD)

F - Change fog

I - Inventory

E - Interact

Q - Drop a object

F1 - Debug

F2 - Hide/Show HUD

F3 - Take a screenshot

F4 - Fly (Creative Mode)

F5 - Switch view

1/2/3/4/5/6/7/8/9 or Mouse wheel scrolling - Select a block

Esc - Pause
